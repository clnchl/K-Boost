# LearningScreen

## Rôle de l'écran

LearningScreen est le point d'entree de la feature Apprentissage.
Il oriente l'utilisateur vers les trois sous-sections MVP:

- Theorie
- Cours
- Notes

## Composants UI

- AppBar avec titre Apprentissage
- Liste de cartes de navigation
- Carte Theorie
- Carte Cours
- Carte Notes

## Interactions utilisateur

- Tap sur Theorie -> ouvre TheoryScreen
- Tap sur Cours -> ouvre CourseScreen
- Tap sur Notes -> ouvre NotesScreen
