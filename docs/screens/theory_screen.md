# TheoryScreen

## Rôle de l'écran

TheoryScreen presente le vocabulaire sous forme de liste de cards.
Chaque mot affiche les informations essentielles pour une lecture rapide.

## Composants UI

- AppBar avec titre Theorie
- Liste de cards de mots
- Card mot avec:
  - word
  - translation
  - romanization

## Interactions utilisateur

- Scroll pour parcourir le vocabulaire
- Tap sur une card -> ouvre WordDetailScreen

## Evolutions futures

- Filtres par categorie grammaticale
- Recherche par mot ou traduction
- Tri par difficulte et lecon
- Mode revision avec repetition espacee
