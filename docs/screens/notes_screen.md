# NotesScreen

## Rôle de l'écran

NotesScreen permet a l'utilisateur de gerer des notes personnelles comme un mini carnet.
Le MVP supporte les operations CRUD en local sur des donnees mockees.

## Composants UI

- AppBar avec titre Notes
- Liste de notes sous forme de cards
- FloatingActionButton pour creer une note
- Dialog de creation/modification
- Dismissible pour suppression rapide

## Interactions utilisateur

- Creer une note
- Modifier une note existante
- Supprimer une note (swipe)
- Consulter la liste des notes

## Evolutions futures

- Synchronisation backend
- Classement par dossiers/tags
- Recherche plein texte
- Enrichissement markdown
